---
title: memories export
description: Export memories to JSON or YAML file.
---

```bash
memories export [options]
```

Export memories to a portable JSON or YAML format. Useful for backup, sharing, or migrating between machines.

## Options

| Option | Description |
|--------|-------------|
| `-o, --output <file>` | Output file path (default: stdout) |
| `-f, --format <format>` | Output format: `json`, `yaml` (default: `json`) |
| `-g, --global` | Export only global memories |
| `--project-only` | Export only project memories |
| `--type <type>` | Filter by type |

## Examples

Export all memories to a file:

```bash
memories export -o backup.json
```

Export project rules as YAML:

```bash
memories export --project-only --type rule -f yaml -o rules.yaml
```

Pipe to another tool:

```bash
memories export --json | jq '.memories | length'
```

## Output Format

The export file contains:

```json
{
  "version": 1,
  "exported_at": "2026-01-15T10:30:00Z",
  "project_id": "github.com/user/repo",
  "memories": [
    {
      "id": "abc123def456",
      "content": "Always use TypeScript strict mode",
      "type": "rule",
      "scope": "project",
      "tags": "typescript,config",
      "created_at": "2026-01-10T08:00:00Z"
    }
  ]
}
```

Exports up to 10,000 memories.
