# Memories Starter Apps

Three minimal starters that demonstrate the same core flow in under 10 minutes:

1. Add memory
2. Search memory
3. Fetch context

## Starters

- [Next.js starter](./nextjs-starter) (`next@16.1.6`)
- [Express starter](./express-starter)
- [Python starter](./python-starter)

Each starter expects:

- `MEMORIES_API_KEY`
- `APP_AUTH_TOKENS` (demo auth mapping: `token|tenantId|userId`)

Optional:

- `MEMORIES_PROJECT_ID`

## Quick pick

- If you want a full web app and API routes: use Next.js.
- If you want a lightweight backend service in Node.js: use Express.
- If you want a backend service in Python: use the FastAPI starter.
